Bookmark RAG system for personal knowledge management.

## Features

- **Vector Database**: Utilizes pinecone for efficient vector storage and retrieval.
- **Embeddings**: Uses GenAI embeddings for high-quality text representation.
- **Extension Support**: Plasmo extension for seamless web clipping.
- **Web Interface**: Next.js for a user-friendly interface.
